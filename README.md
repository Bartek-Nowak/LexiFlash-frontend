# LexiFlash

Aplikacja do nauki języków oparta na fiszkach i systemie powtórek.

## 🚀 Funkcje

- Kategorie i własne zestawy fiszek
- Algorytm powtórek (Leitnera)
- Audio do słówek
- Statystyki postępu

## 🛠️ Tech Stack

- Vue 3, Pinia
- Vue Router
- TailwindCSS

## ▶️ Uruchamianie

```bash
npm install
npm run dev
```
